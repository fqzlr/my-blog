📖 API 接口文档
GET
/api
📋 请求参数
参数名	类型	必填	默认值	说明
server	string	否	tencent	音乐平台，可选值：netease（网易云音乐）、tencent（QQ音乐）
type	string	否	playlist	请求类型，可选值见下方支持矩阵
id	string	否	7326220405	资源ID，如歌单ID、歌曲ID、歌手ID、搜索关键词等
🔢 类型支持矩阵
type 值	说明	netease	tencent
song	单曲信息	✓	✓
playlist	歌单	✓	✓
artist	歌手歌曲	✓	✗
search	搜索	✓	✗
url	播放链接	✓	✓
lrc	歌词	✓	✓
pic	封面图片	✓	✓
