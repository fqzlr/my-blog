## 后台影视管理

- 新增后台影视管理页面（`/admin/bangumi/`），支持搜索并添加影视记录
- **三源搜索聚合**：同时搜索 TMDB、豆瓣、Bangumi，点击结果自动填充表单
- 搜索结果显示封面缩略图、标题、年份、来源标签
- 自动填充：标题、中文名、封面 URL、类型、标签
- 支持手动调整：类型（动漫/电影/电视剧/纪录片/游戏）、状态、评分、标签、备注、时间
- 封面预览功能
- 编辑功能：已有数据支持修改所有字段
- 删除功能
- 影视数据存储在独立 GitHub Gist 中（`bangumi.json`）

## 影视页面数据合并

- 影视游戏页面（`/movies-games/`）自动从 Gist 加载外部影视数据
- 外部数据与本地数据按时间排序混合显示，不分离
- 外部数据自动匹配对应分类 Tab（动漫/电影/电视剧/纪录片/游戏）
- Tab 计数和筛选按钮计数（看过/在看/想看等）自动包含后台数据
- 修复筛选按钮在本地无对应状态数据时被隐藏的问题
- 修复抛弃状态数据不显示的问题
- 修复首次进入页面数据重复插入的问题

## 最近更新组件同步

- 最近更新组件自动加载影视 Gist 数据，与本地数据合并显示
- 数据按时间排序，不与本地数据分离

## 首页跑马灯同步

- 首页跑马灯公告自动统计影视 Gist 中最近 3 天更新的数据
- 在 `[更新]` 部分追加影视更新数量，与本地数据合并显示

## 影视脚本多源降级

- `fetch-media` 脚本新增多源降级策略：TMDB → 豆瓣 → Bangumi → 手动模式
- 解决国内影视、动漫、纪录片在 TMDB 搜不到的问题
- 下载函数添加 User-Agent 和 Referer 头，防止被服务器拒绝
