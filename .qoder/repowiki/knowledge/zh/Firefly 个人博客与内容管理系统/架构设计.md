- **核心框架**：采用 Astro 6.x 作为静态站点生成器，集成 `@astrojs/svelte` 实现 Svelte 5 组件的交互式渲染，利用 `@swup/astro` 实现无刷新页面过渡。
- **内容架构**：通过 `src/content.config.ts` 定义强类型的内容集合（Posts, Bangumi, Moments, Life 等），结合 Zod Schema 进行数据校验，支持 Markdown/MDX 格式。
- **配置体系**：在 `src/config/` 下采用模块化配置设计，通过 `index.ts` 统一导出站点、UI、功能及第三方服务配置，实现关注点分离。
- **编辑工作流**：内置基于 GitHub App 认证的在线编辑系统（`src/utils/editMode.ts`），利用 Web Crypto API 在浏览器端完成 JWT 签名，通过 Cloudflare Workers/Vercel 代理（`src/workers/`）安全地读写 GitHub 仓库内容。
- **插件扩展**：自定义 Remark/Rehype 插件链（`src/plugins/`）处理 Mermaid、PlantUML、图片网格及邮箱保护等高级 Markdown 特性。
